       When starting, i3 looks for configuration files in the following order:

        2. ~/.config/i3/config (or $XDG_CONFIG_HOME/i3/config if set)

        3. /etc/i3/config

        4. /etc/xdg/i3/config (or $XDG_CONFIG_DIRS/i3/config if set)
